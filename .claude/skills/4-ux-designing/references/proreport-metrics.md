# PRO Report Core Metrics & Messaging Guide

**버전**: 1.0.0
**마지막 업데이트**: 2025-11-25
**목적**: PRO Report UI 디자인 시 참고할 핵심 지표 정의 및 메시징 가이드

---

## 1. 핵심 지표 우선순위

### Tier 1: 최우선 (매일 확인) - 상단 핵심 카드 3개

#### 1.1 PRO 매출 기여도
**정의**: PRO 기능(SEO, CRM, 프로모션, 채널)을 통해 발생한 매출 합계

**표시 형식**:
- 큰 숫자: "1,234,567원" (32px, Bold)
- 추이: "지난달 대비 +15.3% ↑" (16px, Success color)
- 세부 breakdown: "SEO 52만 | CRM 38만 | 프로모션 28만 | 채널 5.4만"

**인사이트 메시지 패턴**:
- 기본: "PRO가 만들어준 매출은 {금액}입니다"
- 비교: "지난달 대비 {증감률} {증가/감소}했습니다"
- 넛지: "SEO 최적화로 {금액} 더 늘릴 수 있습니다"

**UI 고려사항**:
- 큰 숫자가 시선을 끌어야 함
- 세부 breakdown은 시각적 bar chart로 표현
- 클릭 시 상세 breakdown 페이지로 이동

#### 1.2 절약 시간/비용
**정의**: PRO 자동화로 절약한 작업 시간과 이를 시급으로 환산한 비용

**표시 형식**:
- 시간: "42.5시간" (32px, Bold)
- 비용: "= 639,412원" (24px, SemiBold)
- 활용 제안: "이 시간으로 신규 상품 85개 기획 가능"

**인사이트 메시지 패턴**:
- 기본: "PRO가 절약해준 시간은 {시간}, 비용은 {금액}입니다"
- 활용: "이 시간으로 {구체적 활동} 가능합니다"
- 예측: "연간 약 {금액} 절약 예상"

**UI 고려사항**:
- 시간을 구체적인 활동으로 변환 (상품 기획, 마케팅 전략 수립 등)
- 비용 절감 효과를 연간으로 환산하여 ROI 강조
- 아이콘: 시계 또는 돈 아이콘

#### 1.3 오늘의 매출/주문
**정의**: 당일 자사몰 + 연동 마켓 매출 및 주문 건수 (실시간)

**표시 형식**:
- 매출: "1,852,000원" (32px, Bold)
- 주문: "(45건)" (20px, Regular)
- 추이: "어제 대비 +8.2% ↑" (16px, Success color)

**인사이트 메시지 패턴**:
- 기본: "오늘 {주문 건수}건, {매출 금액} 달성"
- 비교: "어제 대비 {증감률}"
- 목표: "월 목표까지 {남은 금액}"

**UI 고려사항**:
- 실시간 업데이트 느낌 (10분마다 갱신)
- 자사몰 vs 연동 마켓 비율 시각화
- 아이콘: 그래프 상승 아이콘

---

### Tier 2: 주간 확인

#### 2.1 월간 매출 추이
**정의**: 최근 12개월 매출 추이

**표시 형식**:
- Line chart (12개월)
- 최고 매출 월 강조
- 평균 매출선 표시

#### 2.2 전환율
**정의**: 방문자 대비 구매자 비율

**표시 형식**:
- "8.5%" (24px, Bold)
- PRO 평균 대비 비교
- 업계 평균 대비 비교

#### 2.3 상품별 성과
**정의**: 매출 상위 10개 상품

**표시 형식**:
- 순위 테이블
- 상품 이미지 + 이름 + 매출
- 전월 대비 순위 변동

---

### Tier 3: 월간 확인

#### 3.1 PRO 처리 업무
**정의**: PRO가 자동으로 처리한 작업 내역

**표시 형식**:
- 작업 유형별 건수
- 시간별 활동 그래프
- 가장 많이 사용한 기능 강조

#### 3.2 재고 현황
**정의**: 재고 부족 위험 상품

**표시 형식**:
- 경고 배지 (재고 10개 미만)
- 상품 리스트
- 예상 품절 일자

## 2. 메시징 톤앤매너

### 기본 원칙
- **친근하고 격려적**: "했어요", "할 수 있어요" 사용
- **구체적인 숫자**: "매출 상승" 대신 "매출 52만원 증가"
- **액션 유도**: "확인하기", "시작하기", "최적화하기"

### 감정 표현
- **긍정적 결과**: "축하합니다! 목표 달성했어요"
- **중립적 결과**: "평균 수준을 유지하고 있어요"
- **부정적 결과**: "개선이 필요해요. 함께 해결해볼까요?"

### 예시

#### 좋은 메시징 ✅
- "PRO가 42.5시간 절약해줬어요 = 신규 상품 85개 기획할 시간"
- "이번 달 52만원 더 벌 수 있어요. SEO 최적화로 달성하세요"
- "오늘 45건 주문 달성! 어제보다 8.2% 증가했어요"

#### 나쁜 메시징 ❌
- "절약한 시간은 42.5시간입니다" (추상적)
- "매출을 증대시킬 수 있습니다" (모호함)
- "주문이 증가했습니다" (구체성 부족)

## 3. CTA (Call-to-Action) 가이드

### CTA 위계
1. **Primary CTA**: 가장 중요한 액션 (1개만)
2. **Secondary CTA**: 부가 액션 (1~2개)
3. **Tertiary CTA**: 링크 형태 (여러 개 가능)

### PRO Report 핵심 CTA

#### Primary CTA 예시
- "매출 boost하기" → "SEO 최적화로 52만원 더 벌기"
- "시간 절약 확인하기" → "자동화 설정 완성하기"
- "상세 보기" → "성장 전략 보러가기"

#### Secondary CTA 예시
- "PRO 기능 설정하기"
- "도움말 보기"
- "이전 리포트 보기"

#### Tertiary CTA 예시
- "자세히 알아보기 →"
- "FAQ 보기"
- "고객센터 문의"

### CTA 작성 원칙
- **동사로 시작**: "확인하기", "시작하기", "설정하기"
- **구체적인 혜택**: "52만원 더 벌기", "시간 절약 확인하기"
- **긴급성 부여**: "지금 바로", "오늘만", "한정"

## 4. 상황별 넛지 메시지

### 상황 1: PRO 매출 기여도가 낮을 때
```
PRO 기능을 더 활용해보세요.
SEO 최적화로 평균 52만원 더 벌 수 있어요.

[SEO 설정하기 →]
```

### 상황 2: 절약 시간이 적을 때
```
PRO 자동화 설정을 완성하면
매월 40시간 이상 절약할 수 있어요.

[자동화 완성하기 →]
```

### 상황 3: 매출이 감소했을 때
```
지난달보다 매출이 -8.5% 감소했어요.
프로모션 기능으로 고객 재방문을 유도해보세요.

[프로모션 만들기 →]
```

### 상황 4: 목표 달성했을 때
```
축하합니다! 이번 달 목표 120% 달성했어요.
다음 단계 목표를 설정해보시겠어요?

[목표 설정하기 →]
```

## 5. 데이터 시각화 가이드

### 숫자 포맷팅
- **금액**: "1,234,567원" (천 단위 콤마)
- **비율**: "15.3%" (소수점 1자리)
- **시간**: "42.5시간" (소수점 1자리)
- **건수**: "85건" (정수)

### 색상 의미
- **Green (#00C73C)**: 긍정적 추이, 목표 달성
- **Red (#FF4444)**: 부정적 추이, 경고
- **Blue (#0066FF)**: 중립적 정보, PRO 브랜드
- **Yellow (#FFB800)**: 주의, 개선 필요

### 아이콘 의미
- **▲**: 상승 (긍정적)
- **▼**: 하락 (부정적)
- **━**: 변동 없음 (중립적)
- **→**: 다음 액션, 링크

## 6. 레이아웃 우선순위

### F-Pattern Layout
```
┌─────────────────────────────────┐
│ [핵심 카드 1] [핵심 카드 2] [3] │ ← 시선 집중
│                                 │
│ [매출 추이 그래프]              │ ← 2순위
│                                 │
│ [PRO 매출 기여도 breakdown]     │ ← 3순위
│                                 │
│ [기타 지표]                     │ ← 4순위
└─────────────────────────────────┘
```

### 정보 계층
1. **Hero**: 가장 중요한 숫자 (PRO 매출, 절약 시간)
2. **Context**: 비교 데이터 (추이, 평균)
3. **Detail**: 세부 breakdown
4. **Action**: CTA 버튼

## 7. 모바일 최적화

### 모바일 우선 원칙
- 카드는 세로로 쌓기 (1 column)
- 터치 타겟 ≥ 44×44px
- 폰트 크기 최소 14px
- 스크롤 최소화 (핵심 정보 상단 배치)

### 모바일 전용 패턴
- **Swipeable cards**: 좌우 스와이프로 카드 탐색
- **Collapsible sections**: 상세 정보는 접기/펼치기
- **Sticky header**: 핵심 지표는 항상 상단 고정
