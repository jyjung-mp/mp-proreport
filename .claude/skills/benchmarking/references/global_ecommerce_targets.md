# 글로벌 E-commerce 벤치마킹 대상

## 기본 비교 대상 (우선순위)

### 1. Shopify
- **분류**: E-commerce 플랫폼
- **강점**: 대시보드 UX, 데이터 시각화, 앱 생태계
- **주요 기능**: Shopify Analytics, Reports, Insights
- **공식 사이트**: https://www.shopify.com
- **참고 문서**: Shopify Admin 대시보드, Analytics 기능

### 2. Google (Google Analytics 4, Google Merchant Center)
- **분류**: 데이터 분석 도구
- **강점**: 데이터 정확성, 실시간 분석, 커스터마이징
- **주요 기능**: GA4 Reports, E-commerce Analytics, Conversion Tracking
- **공식 사이트**: https://analytics.google.com, https://merchants.google.com
- **참고 문서**: GA4 공식 문서, Merchant Center Help

### 3. Amazon Seller Central
- **분류**: 마켓플레이스 판매자 도구
- **강점**: 상세 매출 분석, 경쟁사 비교, 광고 성과
- **주요 기능**: Business Reports, Brand Analytics, Advertising Console
- **공식 사이트**: https://sellercentral.amazon.com
- **참고 문서**: Seller Central Help

## 추가 비교 대상 (상황별)

### 4. 쿠팡 파트너스 / 쿠팡 Wing
- **분류**: 한국 마켓플레이스
- **강점**: 로켓배송 데이터, 한국 시장 특화
- **주요 기능**: 판매 현황, 정산 리포트
- **공식 사이트**: https://wing.coupang.com
- **사용 시기**: 한국 시장 비교 필요 시

### 5. 네이버 스마트스토어 센터
- **분류**: 한국 E-commerce 플랫폼
- **강점**: 네이버 검색 연동, 한국 사용자 UX
- **주요 기능**: 스토어 분석, 판매 분석, 광고 분석
- **공식 사이트**: https://sell.smartstore.naver.com
- **사용 시기**: 한국 시장 비교 필요 시

### 6. eBay Seller Hub
- **분류**: 글로벌 마켓플레이스
- **강점**: 글로벌 판매 데이터, 다국가 비교
- **주요 기능**: Performance, Traffic, Sales Reports
- **공식 사이트**: https://www.ebay.com/sh/ovw
- **사용 시기**: 글로벌 마켓 확장 시

## 비교 시 주의사항

1. **최신 정보 필수**: 항상 WebSearch/WebFetch로 최신 상태 확인
2. **Fact-based**: 추측이나 가정 금지, 실제 데이터/스크린샷 기반
3. **비교 대상 제한**: 한 번에 최대 2-3개 대상과만 비교 (정보 과부하 방지)
4. **맥락 고려**: 기업 규모, 타겟 시장, 비즈니스 모델 차이 명시
