## 2. 글로벌 벤치마크 분석

### Shopify Analytics (2024-2025)

**1순위 지표 (매일 확인)**:
- 매출액 (Revenue)
- 주문 건수 (Orders)
- 실시간 트래픽 (Sessions)

**핵심 철학**:
- "실시간으로 빠르게 확인"
- 커스터마이징 가능한 대시보드 (드래그앤드롭)
- 핵심 지표를 한눈에

**참고**: [Shopify New Analytics 2024](https://www.shopify.com/blog/new-analytics)

### Amazon Seller Central (2024-2025)

**1순위 지표 (매일/주간 확인)**:
- **전환율 (Unit Session %)**: 8-12% 정상, 13% 이상 우수
- **재고 성과 지수 (IPI)**: 450 이상 유지, 550+ 목표
- **광고 효율 (TACoS/ACoS)**

**핵심 철학**:
- "비즈니스 의사결정에 직접 영향"
- 예측 지표 (재고 부족 경고, 이탈 위험도)
- 주간 모니터링 필수

### 공통점
- 매출/주문은 기본 중의 기본
- ROI 중심 (광고비 대비 매출, 시간당 수익)
- 실시간 알림 (이상 징후 자동 감지)

## 4. GA4 방법론 기반 지표 설계

### 핵심 지표 1: PRO 매출 기여도

**정의**: PRO 기능(SEO, CRM, 프로모션, 채널)을 통해 발생한 매출 합계

**GA4 원칙 적용**:
- **Event-Based**: 주문 이벤트 중 PRO 기여 필터링
- **Attribution**: 다중 기능 기여도 우선순위 (채널 > 프로모션 > CRM > SEO)
- **User-Centric**: 사용자 여정에서 PRO 터치포인트 추적

**계산식**:
```
PRO_매출 = SEO_매출 + CRM_매출 + 프로모션_매출 + 채널_매출

중복 제거 로직:
IF 주문.채널 IN ['연동마켓'] THEN 채널_매출
ELSE IF 주문.프로모션_코드 NOT NULL THEN 프로모션_매출
ELSE IF 주문.CRM_캠페인_ID NOT NULL THEN CRM_매출
ELSE IF 주문.유입경로 IN ['자연검색', 'SEO'] THEN SEO_매출
ELSE 0
```

**데이터 소스**:
- CA 2.0 API: `/ca2/attribution/traffic-analysis`
- FAS API: `/tasks/pro/report`
- 업데이트: 매월 3일 (D+2)

**인사이트 메시지**:
- 기본: "PRO가 만들어준 매출은 1,234,567원입니다"
- 비교: "지난달 대비 +15.3% 증가했습니다"
- 넛지: "SEO 최적화로 52만원 더 늘릴 수 있습니다"

---

### 핵심 지표 2: 절약 시간/비용

**정의**: PRO 자동화로 절약한 작업 시간과 이를 시급으로 환산한 비용

**계산식**:
```
절약_시간(시간) = Σ(PRO_처리건수 × 작업별_소요시간)

작업별 소요시간 (수동 처리 기준):
- 상품 등록: 10분/건
- CRM 메시지: 3분/건
- 프로모션 설정: 15분/건
- 연동 마켓 주문 처리: 5분/건

절약_비용(원) = 절약_시간 × 시급
시급 = 최저시급 × 1.5 (숙련도 반영, 2025년 기준 10,030원 × 1.5 = 15,045원)
```

**데이터 소스**:
- FAS API: `/tasks/pro/report` (작업 유형별 처리 건수)
- 업데이트: 실시간

**인사이트 메시지**:
- 기본: "PRO가 절약해준 시간은 42.5시간, 비용은 639,412원입니다"
- 활용: "이 시간으로 신규 상품 85개 기획 가능합니다"
- 예측: "연간 약 767만원 절약 예상"

---

### 핵심 지표 3: 오늘의 매출/주문

**정의**: 당일 자사몰 + 연동 마켓 매출 및 주문 건수 (실시간)

**계산식**:
```
오늘_매출 = 자사몰_매출(TODAY) + 연동마켓_매출(TODAY)
오늘_주문 = 자사몰_주문(TODAY) + 연동마켓_주문(TODAY)
```

**데이터 소스**:
- CA 2.0 API: `/ca2/sales/overview` (date=TODAY)
- 마켓플러스 API: `/api/internal/v1/order/daily-sales` (date=TODAY)
- 업데이트: 10분마다

**인사이트 메시지**:
- 기본: "오늘 45건, 1,852,000원 달성"
- 비교: "어제 대비 +8.2%"

## 5. 최종 우선순위 및 권장사항

### Tier 1: 최우선 (매일 확인) - 상단 핵심 카드 3개

| 순위 | 지표 | PRO 차별화 | 벤치마크 | 구현 난이도 |
|------|------|-----------|----------|-------------|
| 1 | PRO 매출 기여도 | ⭐⭐⭐ | - | 높음 |
| 2 | 절약 시간/비용 | ⭐⭐⭐ | - | 중간 |
| 3 | 오늘 매출/주문 | ⭐ | Shopify/Amazon | 낮음 |

### Tier 2: 주간 확인

| 순위 | 지표 | PRO 차별화 | 벤치마크 | 구현 난이도 |
|------|------|-----------|----------|-------------|
| 4 | 월간 매출 추이 | ⭐⭐ | Shopify/Amazon | 낮음 |
| 5 | 전환율 | ⭐⭐ | Shopify/Amazon | 중간 |
| 6 | 상품별 성과 | ⭐⭐ | Shopify/Amazon | 낮음 |

### Tier 3: 월간 확인

| 순위 | 지표 | PRO 차별화 | 벤치마크 | 구현 난이도 |
|------|------|-----------|----------|-------------|
| 7 | PRO 처리 업무 | ⭐⭐⭐ | - | 낮음 |
| 8 | 재고 현황 | ⭐ | Amazon | 높음 |

---

### 구현 우선순위 (단계별)

**Phase 1 (MVP, 1차수)**:
1. PRO 매출 기여도 ← **핵심**
2. 절약 시간/비용 ← **핵심**
3. 오늘 매출/주문 ← 쉬움
4. 월간 매출 추이 ← 쉬움

**Phase 2 (2차수)**:
5. 전환율
6. 상품별 성과
7. PRO 처리 업무

**Phase 3 (3차수)**:
8. 재고 현황

## 7. 핵심 발견사항

### PRO만의 고유 가치 (차별화)
- **PRO 매출 기여도**: "내가 PRO에 돈 낸 게 아깝지 않았다"
- **절약 시간/비용**: "PRO가 나 대신 일해줬다"
- **PRO 처리 업무**: "PRO가 구체적으로 뭘 해줬다"

### 글로벌 표준 (Shopify/Amazon 동일)
- 매출/주문 (실시간)
- 전환율
- 상품별 성과
- 재고 현황

1. **사용자 중심**: "내가 궁금한 것부터"

3. **액션 유도**: "다음에 뭘 해야 할지"
4. **실시간성**: "지금 당장 확인"
